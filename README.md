# Personal Portfolio Website

A fully responsive, multi-page personal portfolio website built for the Web Application Development course at Strathmore University. It presents Salim, a Computer Science student, across nine interlinked pages with a consistent dark theme, a light/dark mode toggle, and several interactive JavaScript features.

## Live Pages

The site has nine interlinked pages, all sharing a common navbar and footer:

- **Home** – hero section with a typing animation, skills preview, stats, and previews of blog posts and testimonials
- **About** – background, education, interests, and fun facts
- **Projects** – project showcase with JavaScript status filtering (All / Completed / In Progress / Planned)
- **Skills** – programming languages with animated progress bars, plus tools and frameworks
- **Services** – services offered and a "How It Works" section
- **Blog** – articles on algorithms, finance, philosophy, and web development, each expandable inline
- **Testimonials** – peer reviews
- **Resume** – education, projects, skills summary, and a journey timeline
- **Contact** – a contact form with JavaScript validation

## Technologies Used

- HTML5
- CSS3 (custom stylesheet)
- JavaScript (ES6)
- Bootstrap 5.3

## JavaScript Features

- Contact form validation (name, email, subject, message) with inline error feedback
- Light/dark mode toggle that remembers the choice between visits
- Project filtering by status
- Animated skill progress bars
- Typing animation in the hero section
- Back-to-top button

## How to Run

Clone the repository and open `index.html` in any modern web browser:

```
git clone https://github.com/8salim3/personal-assignment.git
cd personal-assignment
```

Then open `index.html`, or use the VS Code Live Server extension for automatic reloading.
